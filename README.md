# 🛒 E-Commerce Website (Laravel Project)

## 📌 About the Project
This is a full-stack e-commerce web application built using Laravel. The system allows users to browse products, manage accounts, and interact with categories and products in a structured way.

The project focuses on authentication, product management, and a basic shopping experience.

---

## 🚀 Features

### 👤 User Authentication
- User Registration
- Login / Logout
- Forgot Password functionality

### 🛍️ Product Management
- Add Products
- View Products
- Product categorization

### 📂 Categories
- Create and manage product categories
- Organize products under relevant categories

### 🔐 Security
- Password encryption
- Secure authentication system

---

## 🛠️ Tech Stack
- Laravel 
- PHP
- MySQL Database
- HTML, CSS, Bootstrap 
- JavaScript
